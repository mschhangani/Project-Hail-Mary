# Project Hail Mary - Journey to Tau Ceti

A single-file interactive science exhibit about what it would actually take — 
and what it would actually feel like — to cross 11.9 light-years to Tau Ceti.

Scroll to fly the mission. Every number the instrument dock shows is computed 
live from the relativistic rocket equations: proper time, Lorentz factor, 
length contraction, communication lag, the works.

## What's inside

Nine chapters — departure, constant-thrust acceleration, a hands-on relativity 
lab with a velocity slider, communication delay simulation, the interstellar 
medium at near-light speed, an interactive ship cutaway, hibernation science, 
a pannable star map, and a cinematic arrival.

## Mission profile

| | |
|---|---|
| Destination | Tau Ceti (11.9 ly) |
| Thrust | 1.5 g constant acceleration |
| Peak velocity | 0.995 c |
| Peak Lorentz factor | γ = 10.21 |
| Crew time elapsed | 3.89 years |
| Earth time elapsed | 13.13 years |

## Run it

Open the file in any modern browser. No build step, no server, no install.

Inspired by Andy Weir's *Project Hail Mary*.
